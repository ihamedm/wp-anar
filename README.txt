پلاگین انار محصولاتی که در پنل انار۳۶۰ شما اضافه شده اند را به عنوان محصول ووکامرسی به سایت شما اضافه می کند.

== Description ==

پلاگین انار محصولاتی که در پنل انار۳۶۰ شما اضافه شده اند را به عنوان محصول ووکامرسی به سایت شما اضافه می کند.


== Installation ==

مراحل نصب پلاگین انار کاملا مشابه سایر پلاگین هاست. بعد از نصب لازم است وارد بخش تنظیمات پلاگین شوید و توکن دریافتی خود از پشتیبانی انار را وارد کنید تا امکان استفاده از پلاگین برای شما فعال شود.

== Frequently Asked Questions ==

= پیش نیاز نصب پلاگین انار چیست؟ =

برای نصب پلاگین انار لازم است پلاگین فروشگاه ساز ووکامرس نصب باشد.

= دریافت محصولات انار چقدر طول می کشد؟ =

بعد از انجام مراحل دریافت محصولات انار هر دقیقه ۳۰ محصول از سرور انار دریافت و به محصولات ووکامرسی شما اضافه می شود.

= چرا دریافت محصولات با سرعت پایین انجام می شود؟ =

با توجه به اینکه لازم است پلاگین در هاست های با منابع پایین هم بدرستی کار کند لازم است این فرآیند به کمترین فشار در پس زمینه انجام شود. در نسخه های بعد امکان دریافت محصولات با سرعت بالاتر برای هاست های پرقدرت تر افزوده می شود.

= اگر محصولات جدیدی به اکانت انارم اضافه کردم چطور به وب سایت افزوده می شوند؟ =

لازم است مجدد مراحل افزودن محصولات را طی کنید. هیچ مشکلی برای محصولاتی که قبلا اضافه شده اند پیش نمی آید و صرفا محصولات جدید به وب سایت اضافه می شوند.
